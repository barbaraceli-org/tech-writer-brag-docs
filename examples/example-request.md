## With extra context

Include work not in Jira or GitHub:

```
Weekly brag doc for last week.

Also include:
- Presented "Docs best practices" at team sync
- Mentored new hire on first PR
- Updated team style guide
```

---

## Tips

- **Be explicit about period** if needed: "last week", "February 2026", "Q1 2026".
- **Add context** for mentoring, talks, process changes, or one-off wins.
- Jira and (if configured) GitHub are pulled automatically; no need to list issues or PRs.
